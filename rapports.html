Rapport sur Brave Browser


C'est le browser que j'ai utilisé pour vérifier et coder mon TP, alors tout marche comme je le veux. Il y a aucune photo manquante, couleur ou code CSS qui ne fonctionne pas comme je l'ai souhaité, les autres rapports seront peut-être plus intéressants.

Rapport sur Chrome


Sur chrome, il n'y a aucun problème avec mon site web, chrome utilise la même base que Brave alors les résultats sont très similaires. Chrome est également à jour avec le code HTML et CSS alors c'est normal qu'il n'y ai pas de problèmes.

Rapport sur Internet Explorer


Sur Explorer, il n'y a aucun problème majeur, le seul problème apparent est que la police monospace ne fonctionne pas et que les photos sont plus pixelisés que sur mes autres navigateurs. Sinon, tout fonctionne parfaitement après avoir vérifier chaque page une par une.

Fait par Alexandre Brunet Breton, Février 2021
